# OverBooked – Reading Club Website

A responsive and modern website developed for **OverBooked**, the Reading Club of IIIT Guwahati.

## About

OverBooked is the official Reading Club of IIIT Guwahati. This website was designed and developed to showcase the club's mission, featured books, events, and community while providing an engaging experience for students and book enthusiasts.

## Features

- Responsive landing page
- Hero section
- About the club
- Featured books
- Upcoming events
- Why Join Us section
- Coordinators section
- Testimonials
- Contact section

## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
